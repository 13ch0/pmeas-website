Welcome to the homepage of the Portable Multi-Audio Effects Software System (PMEAS)! PMEAS is an open source software package that allows you to modulate audio played from an instrument. At present, there are seven unique effects that can be applied to the audio stream. These effects can be stacked on top of one to create a plethora of unique sound combinations. The system also comes with an optional loop machine that allows you to playback (and stack) sounds you can record. The video provided below is a great demonstration of many of the functionalities in action.


Optimized for Speed

The PMEAS system is optimized for speed. Modulate audio from your instrument with latency that does not exceed 20ms.


Open Source

This project was developed to give back to the wonderful community of musicians by providing all components of the PMEAS system at absolutely no cost.


Cross Platform

Run the graphical interface of the application in multiple platforms! Able to run on Windows, Linux, OS X, and even Android applications!
